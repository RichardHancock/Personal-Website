Millionaire Quiz

Stage:

Working Prototype

Languages Used:

Visual Basic
  • Source
  • Download

Description:

A complex quiz based on the British version of TV quiz "Who Wants to Be a Millionaire?". Choosing a pre-established concept allowed me to get right into the code with some basic assets. I have a working prototype with question progression, a back-end money system and the 50-50 lifeline. It is currently still being made, but due to lack of time I haven't been able to get much more done.


Due to copyright reasons it will probably never be released in any form, but it was still good practice.

Pong

Stage:

Completed

Languages Used:

Visual Basic
  • Source
  • Download

Description:

A basic clone of Pong that is one player vs computer. Very simple idea that allowed me to practice collision detection and test the limits of Visual Basic (Pretty Low). It is fully functional but has an annoying slowness to it. I hope to try and create this again later in a different language.


Disclaimer: Pong is the copyright of Atari, This version was created for educational purposes and will not be released publicly.

Basic Battle Prototype

Stage:

Working, Improvements Planned

Languages Used:

Visual Basic
  • Source
  • Download

Description:

After playing way too much Pokemon I decided to make a simple battle system prototype to see how easy it would be. I made this basic version with very quick graphics which turned out reasonably well. After making a basic version with health and move sets it seemed pretty hard to lose against the random move select AI. So in later versions I plan to add a AP/fatigue/mana type resource to try and balance things out, along with some better graphics.
