These are some helpful websites I am using for software development and to learn programming:

Cloud Storage

  • Dropbox
  • Google Drive
  • Pocket - Great service for saving links for later, accessible from most devices.
  • SkyDrive

Programming

  • GitHub - Git version control repository host, private repo's cost money.
  • The New Boston - Varied tutorials about programming and other computer programs.
  • Udacity - Intro to Computer Science Course with very well produced videos and programming challenges. Uses Python.

Web Development

  • Codecademy - Good practice in web development languages, JavaScript and Python.
  • HTML Dog - Good reference for web development.
  • Web Platform - Website full of references and tutorials from all major web companies collaborating.

Other

  • Trello - Good project management tool.
